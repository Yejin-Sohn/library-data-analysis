# 📊 최종 분석 보고서

**프로젝트**: 도서관 자리는 항상 부족한데, 왜 빈자리가 보일까?
**기관**: 한국외국어대학교 제1열람실
**기간**: 2026년 3월 ~ 6월 (3개월)
**팀**: 데이터사이언스 5팀 (6명)

---

## 📌 Executive Summary

**핵심 결론**:
도서관 좌석 부족 문제는 **절대적인 좌석 수 부족이 아닌, 이용 편중·시간 집중·장시간 점유에서 비롯**된다는 점을 데이터로 확인했습니다. 따라서 좌석 증설보다 **기존 좌석의 활용도를 높이는 운영 정책**이 더 효과적입니다.

**3가지 정책 제안**:
1. **자동 반납 강화** (90분 → 60분)
2. **저이용 좌석 환경 개선** (출입문 인근 시선 차단)
3. **노쇼 관리** (연속 IN 차단 알고리즘)

→ 모두 **추가 좌석 설치 없이 기존 자원 활용도를 높이는 방안**

---

## 1. 문제 정의

도서관에 가면 자리는 항상 부족한데, 빈자리는 곳곳에 보입니다.

이 모순을 데이터로 검증하기 위해 다음 3가지 질문을 설정했습니다:

| 세부 질문 |
|---|
| ① 좌석 이용은 모든 자리에 고르게 분포하는가? |
| ② 특정 좌석이 반복적으로 더 많이 사용되는가? |
| ③ 시간대별로 좌석 점유 패턴은 어떻게 달라지는가? |

→ **분석 목표**: 좌석 부족 문제를 공간·시간·이용 행태·학생 인식 관점에서 분석

---

## 2. 사용 데이터 (3종 결합)

| 데이터 | 규모 | 수집 방법 |
|---|---|---|
| 좌석 이용 로그 | 665,156건 | 자동 크롤링 (10분 단위 스냅샷) |
| 도서관 배치도 | 478개 좌석 + 공간 요소 | 현장 실측 + 좌표 매핑 |
| 이용자 설문 | 160명 응답 | Google Forms 설문 |

→ **실측(행동) + 공간(배치) + 인식(설문) 3중 결합**으로 좌석 부족 현상의 **원인까지** 규명

---

## 3. 분석 흐름

```
실측 데이터 (행동) → 공간적 원인 → 시간·회전율 문제 → 학생 인식 → 정책 제안
```

## 4. 분석 결과

### 🔍 A. 좌석 이용 편중 분석

**결론: 좌석 이용은 균등하지 않고, 특정 좌석에 집중된다.**

| 지표 | 결과 | 해석 |
|---|---|---|
| 카이제곱 검정 | χ² = 170,293.6, p < .001 | 균일 이용 가설 강력 기각 |
| Gini 계수 | 0.268 | 중간 수준 편중 |
| 정규화 엔트로피 | 0.980 | - |
| 좌석별 점유율 범위 | 0.0% ~ 88.6% | 큰 격차 |
| 과이용 좌석 (표준화 잔차 \|z\|>2) | 171개 | 기대보다 많이 쓰임 |
| 저이용 좌석 (표준화 잔차 \|z\|<-2) | 275개 | 기대보다 적게 쓰임 |
| 좌석번호 vs 점유율 상관 (Spearman) | ρ = 0.005 | 좌석번호 자체와는 무관 |

**→ 좌석 이용 편중은 우연이 아닌 통계적으로 유의한 구조적 현상.**
**→ 편중은 좌석번호가 아니라 위치·환경 요인에 의해 발생.**

---

### 🔍 B. 좌석과 공간 요소의 위치관계

**결론: 위치 요인이 좌석 선택을 결정한다.**

| 공간 요인 | Spearman ρ | p-value | 해석 |
|---|---|---|---|
| **가장자리 점수** | **+0.45** | < .001 | 🟢 벽면·외곽 좌석 선호 (가장 강함) |
| **출입문 거리** | **+0.28** | < .001 | 🟢 출입문에서 멀수록 선호 |
| **창문 거리** | **-0.22** | < .001 | 🟢 창문에 가까울수록 선호 |
| 기둥 거리 | +0.05 | 0.291 | ❌ 유의하지 않음 |

**→ 이용자는 "심리적 안정감과 독립성이 높은 공간"을 선호한다.**
- 출입문 인근 = 이동·소음·시선 노출 회피
- 창가 = 채광·개방감·벽면 안정감
- 가장자리 = 독립성·심리적 안정감

---

### 🔍 C. 시간대별 점유율 차이

**결론: 인기 좌석은 전체 평균보다 빨리 포화된다.**

| 시간대 | 전체 평균 점유율 | 상위 30석 점유율 | 특징 |
|---|---|---|---|
| 오전 (5~12시) | 8.6% | 40% | 유휴 좌석 많음 |
| 오후 (12~19시) | 34.8% | **80.0%** | 인기 좌석 조기 포화 |
| 저녁 (19~익일 1시) | 38.1% | 60% | 전체 혼잡 최고 |

**통계 검정**:
- Kruskal-Wallis H = 2,564.55 (p < .001) → 시간대 간 차이 유의
- Mann-Whitney U (Bonferroni 보정):
  - 오전 vs 오후: 유의 ✅
  - 오전 vs 저녁: 유의 ✅
  - 오후 vs 저녁: 비유의 ❌ → 오후 이후 혼잡 지속

**→ 실제 좌석 경쟁은 "전체 혼잡 최고 시점"이 아닌 "인기 좌석 조기 포화 시점".**

---

### 🔍 D. 시험 기간 vs 평시 비교

**결론: 시험기간은 평시의 2.6~2.9배 점유.**

| 기간 | 평균 점유율 | 평시 대비 |
|---|---|---|
| 평시 | 16.1% | 1.0배 |
| 시험 직전 | 42.7% | **2.65배** |
| 시험 기간 | 47.2% | **2.93배** |

**통계 검정**:
- Kruskal-Wallis H = 1,112.15 (p < .001) → 기간 간 차이 유의

**패턴 변화**:
- 평시: 오후 피크 후 저녁 완만 감소
- 시험기간: **저녁(19시 이후) 60~80% 고점유 지속** → 밤샘 이용·장시간 체류 증가

**→ 시험기간은 좌석 수요 폭증 + 회전율 저하 동시 발생.**

---

### 🔍 E. 설문 결합 분석 (160명)

**결론: "점유 ≠ 사용" 격차가 좌석 부족의 진짜 원인.**

#### 핵심 발견

| 항목 | 결과 | 시사점 |
|---|---|---|
| 30분 이상 자리비움 | **43.8%** | 점유와 실제 사용 격차 큼 |
| 자리비움 허용 인식 | **58.8%** | 다수가 문제로 인식하지 않음 |
| 시스템 만족도 | 3.71/5 | 보통 수준 |
| 자동 반납 적정시간 1위 | **60분 (36.2%)** | 90분은 길다고 인식 |
| 개선안 1위 (가중점수) | **미사용 좌석 자동 반납 (243점)** | 정책 수요 명확 |
| 시스템 불편 1위 | **예약 후 노쇼 (48.1%)** | 노쇼 문제 심각 |

**→ 이용자들도 "빈자리는 보이는데 못 쓰는 문제"를 인지하고 있으며, 자동 반납 강화 정책에 대한 수용도가 높다.**

---

## 5. 실측 + 설문 결합 해석

| 실측 발견 | 설문 설명 |
|---|---|
| 좌석 이용 편중 (χ²=170,293.6) | 이용자는 위치·환경을 우선시 (콘센트는 모든 좌석에 있음) → **편중 원인 = 환경 선호** |
| 위치 요인 유의 (가장자리 ρ=0.45) | 시선·주변 환경 중요시 → **저선호 좌석은 환경 보완 필요** |
| 점유 여부만 측정 가능 | 30분+ 자리비움 43.8% → **"점유 ≠ 사용" 격차 확인** |
| 좌석 회전율 저하 | 자동 반납 60분 선호 (36.2%) → **자동 반납 강화 정책 근거** |

---

## 6. 정책 제안 3가지

> 모두 **추가 좌석 설치 없이 기존 자원의 활용도를 높이는 방안**입니다.

### ① 자동 반납 강화 — 미사용 좌석 회수

**문제**:
- 응답자 43.8%가 30분 이상 자리비움
- 58.8%가 이를 문제로 인식하지 않음
- 점유와 실제 사용 격차 존재

**설문 근거**:
- 자동 반납 적정시간 60분(36.2%) 1위
- 개선 우선순위 1위 (가중점수 243점)

**제안**:
- 현재 **90분 무사용 자동 반납 → 60분으로 단축**
- 특히 시험기간 우선 적용 (회전율 확보 필수)

---

### ② 저이용 좌석 환경 개선 — 위치 편중 완화

**문제**:
- 출입문 인근·중앙 좌석의 지속적 저이용
- 좌석 부족보다 **특정 좌석으로의 이용 집중**이 핵심

**설문 근거**:
- 콘센트 중요도는 높게 평가됨 (3.88/5)
- 그러나 콘센트는 이미 전 좌석에 동일 제공
- → **시설보다 환경 문제**

**제안**:
- 출입문 인근 저이용 좌석에 **파티션·가벽·식물 등 시선 차단 요소** 설치
- 신규 좌석 증설보다 **기존 좌석 활용률 개선**에 집중

---

### ③ 노쇼 관리 — 예약 효율 향상

**문제**:
- 시스템 불편사항 1위: **예약 후 노쇼** (48.1%)
- 게이트는 입·출입 구분 없이 모두 IN으로 처리
- 예약만 유지된 채 실제 사용되지 않는 좌석 발생

**설문 근거**:
- 실시간 좌석 현황 제공: 225점
- 노쇼 패널티 도입: 200점

**제안**:
- 노쇼를 **"게이트 인으로 속인 후 노쇼"** 와 **"실제 게이트 인 후 노쇼"** 로 구분
- 데이터 기반 관리 가능한 전자에 집중
- **연속 IN 차단 알고리즘** 적용:
  - OUT → IN : 정상
  - IN → IN (15분 이상 간격) : 비정상 로그
  - 반복 발생 : 경고 → 좌석 자동 반납 → 예약 제한

## 7. 결론 및 시사점

### 핵심 인사이트

1. **좌석 부족 ≠ 좌석 수 부족**
   - 실제 원인은 이용 편중, 시간 집중, 장시간 점유
2. **위치가 선호를 결정**
   - 시설보다 공간 환경(가장자리·출입문 거리·창문)이 중요
3. **점유 ≠ 사용**
   - 자리비움이 좌석 회전율 저하의 주범
4. **정책 수요 명확**
   - 이용자도 자동 반납 강화를 원함

### 실무적 의의

- 시설 확충 없이도 좌석 효율 개선 가능
- **데이터 기반 운영 정책의 구체적 기준** 제공
  - 자동 반납 시간(60분)
  - 환경 개선 대상(출입문 인근 저이용 좌석)
  - 노쇼 관리 알고리즘 방향

### 학술적 의의

- **실측 + 인식 결합 분석**의 유효성 입증
- 공간 변수 수치화(가장자리 점수)의 통계적 의미 확인
- 통계 검정 다중 적용(카이제곱·Spearman·Kruskal-Wallis·Mann-Whitney U)
