## 📁 데이터 파일 (3종)

### 1. `library_seats.csv` — 좌석 이용 로그
**좌석별 점유 여부를 10분 단위로 기록한 시계열 데이터**

| 항목 | 내용 |
|---|---|
| 출처 | 한국외국어대학교 도서관 시스템 |
| 수집 방법 | 도서관 홈페이지 자동 크롤링 스크립트 (팀원 작성) |
| 수집 주기 | **10분 단위 스냅샷** |
| 수집 시간 | 오전 5시 ~ 익일 오전 1시 |
| 수집 기간 | 2026년 4월 11일 ~ 5월 24일 |
| 총 관측 수 | **665,156건** |
| 분석 좌석 수 | 478개 (활성 좌석) |

**주요 컬럼**:
- `collected_at`: 데이터 수집 시각
- `seats_data`: 좌석별 정보 (JSON 문자열)
  - `code`: 좌석 번호
  - `isActive`: 좌석 활성화 여부
  - `isOccupied`: 점유 여부
  - `chargeTime`: 이용 시간 (분)
  - `remainingTime`: 남은 시간 (분)
  - `room.id`: 열람실 ID

---

### 2. `layout_data.json` — 도서관 배치도 데이터
**도서관 제1열람실의 좌석·공간 요소를 좌표로 매핑한 데이터**

| 항목 | 내용 |
|---|---|
| 출처 | 도서관 배치도 + 직접 현장 실측 |
| 수집 방법 | 팀 공동 작업 (현장 실측 → 좌표 공간 매핑) |
| 구성 요소 | 좌석·문·창문·벽·기둥·통로·모션데스크 |

**주요 변수**:
- `x`, `y`: 위치 좌표
- `width`, `height`: 크기
- `rotation`: 회전각
- `type`: 요소 유형 (seat/cubicle/disabled/motion/door/window/pillar/wall)

**색상 범례**:
| 색상 | 유형 |
|---|---|
| 연한 민트색 (`#d1f2eb`) | 개방형 좌석 |
| 하늘색 (`#a9cce3`) | 칸막이 좌석 |
| 보라색 (`#d7bde2`) | 장애인 좌석 |
| 연노랑색 (`#fcf3cf`) | 모션데스크 |
| 연빨강색 (`#f5b7b1`) | 출입문 |
| 민트색 (`#a3e4d7`) | 창문 |
| 회색 (`#aeb6bf`) | 기둥 |

### 3. `library_survey.csv` — 이용자 설문 데이터
**한국외국어대학교 학생 대상 도서관 이용 행태·인식 설문**

| 항목 | 내용 |
|---|---|
| 출처 | Google Forms 설문 |
| 대상 | 한국외국어대학교 학생 |
| 응답 수 | **160명** |
| 수집 기간 | 2026년 5월 |
| 설문 설계 | **본인 직접 설계** (실측 분석 결과와 결합 가능하도록 문항 기획) |
| 응답 수집 | 팀 공동 (설문 배포·홍보) |

**설문 구성** (주요 질문):

#### A. 응답자 프로파일
- Q2. 도서관 이용 빈도
- Q3. 주 이용 시간대 (복수응답)
- Q4. 1회 평균 체류시간

#### B. 좌석·환경 선호
- Q7. 선호 좌석 종류
- Q8. 선호 이유
- Q9~Q12. 환경 요인 중요도 (콘센트·창가·시선·통로 거리 등)

#### C. 자리비움
- 자리비움 시간 분포
- 자리비움 발생 이유
- 자리비움 허용 인식

#### D. 시스템 만족도·개선안
- Q19. 시스템 만족도 (5점 척도)
- Q20. 주요 불편 사항
- Q21. 자동 반납 적정 시간
- Q22~24. 개선안 우선순위 (가중점수: 1순위×3 + 2순위×2 + 3순위×1)

## 🔍 분석에 사용된 데이터 결합 방식

```
[실측 행동 데이터]              [공간 메타데이터]              [사용자 인식 데이터]
library_seats.csv      ─┐    layout_data.json      ─┐    library_survey.csv
(665,156건 점유 관측)      │   (좌석·공간 좌표)         │   (160명 응답)
                          │                            │
                          └────────────┬───────────────┘
                                       ↓
                              [통합 분석 데이터셋]
                              df_seats (좌석별 통계)
                              + spatial features (위치 요인)
                              + survey responses (인식 데이터)
                                       ↓
                              [실측 + 인식 결합 인사이트]
```

## 📜 데이터 윤리·개인정보

- ✅ 모든 설문 응답은 **익명** 수집
- ✅ 도서관 좌석 데이터는 **공개 시스템에서 수집** (개인 식별 정보 없음)
- ✅ 분석 결과 외 원본 데이터는 외부 공유하지 않음
- ✅ 분석 종료 후 데이터는 안전하게 폐기
